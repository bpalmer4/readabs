abs_catalogue

Usage

abs_catalogue()-> DataFrame

What it does

This function returns the ABS Catalogue information as a pandas DataFrame.

Parameters

The function does not take any arguments.

Return values

The function returns a pandas DataFrame. The index for the DataFrame are the ABS Catalogue Numbers (which are strings). Each row in the DataFrame relates to an ABS Catalogue Number.

Example

import readabs as ra
from pandas import DataFrame
catalogue: DataFrame = ra.abs_catalogue()
Return to the Index
