read_abs_series

Usage

read_abs_series(cat, series_id, **kwargs)-> tuple[DataFrame, DataFrame]

What it does

For the selected ABS Series IDs, this function returns a tuple of two DataFrames. The first DataFrame contains the actual data. The second DataFrame contains the metadata.

Parameters

cat: str -- the ABS Catalogue Number for the data to be downloaded and made available by this function. This argument must be specified in the function call.

series_id: str | Sequence[str] -- an individual series identifier, or a list of identifiers. This argument must be specified in the function call.

Optional keyword arguments

Uses the same optional keyword arguments as read_abs_cat.

Return values

The function returns a tuple of two DataFrames. The first is the actual data associated with the requested series identifiers. The second item is a DataFrame of ABS metadata for the selected series.

Example

import readabs as ra
from pandas import DataFrame
cat_num = "6202.0"  # The ABS labour force survey
unemployment_rate = "A84423050A"  # Australia, all persons, seasonally adjusted, unemployment rate
seo = "6202001"  # The ABS table name
data: tuple[DataFrame, DataFrame] = ra.read_abs_series(cat=cat_num, series_id=unemployment_rate, single_excel_only=seo)
